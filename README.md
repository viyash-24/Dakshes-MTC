# Dakshes Multi Trade Center

A modern online dress shop web application built using Next.js where users can browse products, select size and quantity, and place orders via WhatsApp. Admin can add and manage products with image links.

---

##  Features

- View products with images  
- Select size and quantity  
- Order via WhatsApp  
- Admin can add/edit products  
- Simple and user-friendly UI  

---

##  How It Works

1. User opens the website  
2. Browses available products  
3. Selects a product  
4. Chooses size and quantity  
5. Clicks "Order via WhatsApp"  
6. A pre-filled message is generated  
7. User is redirected to WhatsApp and sends the order to the owner  

---

##  Tech Stack

- Next.js  
- React  
- Tailwind CSS  
- WhatsApp URL API  

---
## WhatsApp Order

Orders are sent using a WhatsApp link with a pre-filled message including:

- Product name  
- Size  
- Quantity
